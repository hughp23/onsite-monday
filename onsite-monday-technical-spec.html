Executive Summary

Onsite Monday is a two-sided marketplace connecting tradespeople with job posters in the UK construction and trades sector. The platform enables tradespeople to find work, get hired, and get paid — all through a single, trusted app. This document outlines two development paths: an investor-ready prototype (MVP) and the full production application.

Technology Stack
Recommended Technology Stack

Each platform has been chosen for cost-efficiency, scalability, and suitability for a UK-based trades marketplace.

📱
React Native
Mobile App (iOS & Android)
Cross-platform framework saving 30–40% versus building separate native apps. Single codebase maintained by one team. Used by major apps including Facebook, Instagram, and Shopify.
⚡
.NET 8 (ASP.NET Core)
Backend API
Enterprise-grade, high-performance framework with strong typing via C#. Excellent for building secure, scalable REST APIs. Built-in dependency injection, authentication middleware, and Entity Framework Core for database access. Strong ecosystem for payment and cloud integrations.
🗄️
PostgreSQL
Primary Database (AWS RDS)
Robust relational database for user profiles, jobs, reviews, and transactions. Supports complex queries needed for search filtering and reporting.
☁️
AWS (Amazon Web Services)
Cloud Hosting & Infrastructure
EC2 for compute, RDS for database, S3 for file storage, CloudFront CDN. UK region (eu-west-2 London) ensures GDPR-compliant data residency. Pay-as-you-go pricing.
💳
Stripe Connect
Payments & Escrow-Like Holding
Industry-standard marketplace payment solution. Supports delayed payouts (up to 90 days) for escrow-like fund holding. Handles KYC, PCI compliance, and sanctions screening. 1.4% + 20p per UK card transaction.
🔐
Firebase Auth
Authentication
Secure sign-up/sign-in with email and password. Supports biometric login, social auth (Google/Apple), and magic links. Free tier covers early-stage usage.
💬
Stream / SendBird
In-App Messaging
Real-time chat between job posters and applicants. Pre-built UI components for rapid integration. Supports file sharing, read receipts, and typing indicators.
🔔
Firebase Cloud Messaging
Push Notifications
Job alerts, application updates, payment confirmations. Free tier covers early-stage volume. Works across both iOS and Android.
📍
Google Maps API
Location & Distance
Location-based job search, travel radius slider, distance calculations for matching tradespeople to nearby jobs. Generous free tier.
📧
SendGrid + Twilio
Email & SMS Notifications
Automated transactional emails (job confirmation, reviews, receipts). Twilio for SMS alerts. Combined solution for all outbound comms.

Payment Flow – Stripe Connect
📋
1
Job Posted
Client creates a job listing with day rate and duration. Total cost is calculated.
🤝
2
Tradesperson Accepted
Match confirmed. Client authorises payment via Stripe Checkout.
🔒
3
Funds Held
Payment captured and held in Stripe Connect platform balance. Not yet paid to tradesperson.
✅
4
Job Completed
Tradesperson marks job as complete. Client is prompted to confirm and review.
⭐
5
Mandatory Review
Client MUST leave a star rating + text review to release payment.
💰
6
Payout Released
Funds released to tradesperson's bank. Timing depends on tier (Bronze: 30 days, Silver: 14, Gold: 7, Platinum: 2).
Stripe Connect Fees (UK): 1.4% + 20p per UK card transaction. No monthly fee. Stripe handles PCI compliance, KYC verification, and sanctions screening. The platform can add its own commission on top of Stripe's fee.

Risks & Next Steps
Key Risks & Considerations
Payment Regulation (PSD2/FCA)
High
Using Stripe Connect means Stripe acts as the regulated entity, reducing compliance burden. Ensure escrow-like holding model is clearly documented with Stripe.
Data Protection (GDPR)
High
Processing tradesperson data requires full GDPR compliance. Implement data export, deletion requests, consent management, and privacy notices. Host in AWS London region.
Marketplace Cold Start
Medium
Two-sided marketplace challenge: you need both job posters AND tradespeople. Consider seeding with manual outreach and a referral programme in target cities (York, Leeds).
App Store Approval
Medium
Both Apple and Google have specific rules for marketplace apps. Ensure subscriptions go through their billing systems where required.
Insurance & Liability
Medium
As a platform connecting trades, clarify your liability position. Terms of service should state Onsite Monday is a marketplace, not an employer. Seek legal advice.
Scalability
Low (early)
The recommended stack (AWS + PostgreSQL + Redis) scales well. Architectural decisions made now determine how easily you grow from 500 to 50,000 users.
